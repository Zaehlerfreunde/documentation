# Subdomain der Web-App

Sie haben die Möglichkeit Ihre Lösung entweder über eine Subdomain, von uns oder von Ihnen, zugänglich zu machen.

## Zählerfreunde Subdomain

Die Domain der Web-App kann ganz einfach über das Admin-Portal auf eine Zählerfreunde Subdomain (xyz.zaehlerfreunde.de) eingestellt werden. Einfach im Admin Portal die [Subdomain Einstellungen](https://app.admin.zaehlerfreunde.de/settings/whitelabel) öffnen und die Subdomain Ihrer Wahl konfigurieren.

Falls Sie die Web-App über Ihre eigene Domain zur Verfügung stellen möchten, müssen Sie Ihre eigene Subdomain konfigurieren.

## Eigen Subdomain verwenden

Damit Sie eine Subdomain Ihres Unternehmens verwenden können, muss diese auf unsere Web-App zeigen. Hierfür benötigen Sie Zugrifff auf die DNS-Einstellungen ihrer Domain (z.B. IONOS, oder AWS Route 53).

1. Überlegen Sie sich, welche Subdomain Sie für das Portal möchten.
> Wichtig ist, dass es eine eigene Subdomain ist (z.B. energieportal.ihr-unternehmen.de) und **nicht** nur ein separater Pfad (ihr-unternehmen.de/energieportal)

2. Kontaktieren Sie [uns](mailto:partnersupport@zaehlerfreunde.com) und lassen Sie uns Ihre Wunsch-Domain wissen.
3. Hinterlegen Sie die zwei DNS Einträge welche wir Ihnen zukommen lassen. Der erste Eintrag erlaubt es uns einen SSL-Schlüssel zu erstellen für eine sichere HTTPS-Verbindung. Der zweite Eintrag stellt sicher, dass die Subdomain dann auf unsere Web-App zeigt.
