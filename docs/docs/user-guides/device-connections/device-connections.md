# Geräteanbindungen

Diese Sektion beschreibt kurz, wie Geräte mit der Plattform verbunden werden.

Sie enthält Informationen zu Voraussetzungen, Gerätemeldung und -registrierung,
Authentifizierungs- und Autorisierungsverfahren, unterstützten Protokollen sowie
grundlegenden Konfigurations- und Fehlerbehebungsanweisungen.
